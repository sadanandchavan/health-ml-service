Explanation of Readings

Glucose: Normal fasting: 70-100 mg/dL, post-meal: < 140 mg/dL

Blood Pressure: Normal: 90/60 – 120/80 mmHg

The ML model predicts the probability of high-risk conditions based on these readings.

How Scores Are Calculated

The ML model predicts a probability score between 0 and 1:

  • Score close to 1 → High risk
  • Score close to 0 → Low risk

These scores are mapped to categories: Low, Moderate, High risk.

API Details

  • POST /predict — Predict risk for a single reading. Provide JSON with value, systolic, diastolic, type, context.
  • POST /predict_batch — Predict risk for multiple readings. Provide list of readings.
  • POST /alert — Generate alert for a single reading. Returns risk score and alert message.
  • GET /docs — Interactive Swagger UI to test all endpoints and see request/response models.
